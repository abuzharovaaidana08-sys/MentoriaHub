# 🎓 Mentoria Hub — EdTech Platform MVP

> Единая платформа для поиска образовательных возможностей и асинхронного обучения учеников 8–11 классов.

## 🚀 Быстрый запуск

Просто открой `mentoria-hub.html` в браузере — больше ничего не нужно!

Или задеплой на GitHub Pages / Netlify (drag & drop файл).

---

## ✅ Что реализовано

### Обязательные функции (MVP Checklist)
- [x] **Главная страница** — hero, ценностное предложение, статистика, фичи, CTA-кнопки
- [x] **Регистрация / Вход** — localStorage auth, сессия сохраняется между посещениями
- [x] **Профиль** — класс, интересы (STEM, Бизнес, Олимпиады и др.), цели
- [x] **Каталог возможностей** — 10 карточек с названием, описанием, дедлайном, форматом
- [x] **Фильтры** — по направлению, классу, формату, поиск по ключевым словам
- [x] **Избранное / Сохранение** — сохранение и удаление из избранного
- [x] **5 курсов** с уроками — Математика, Английский, Программирование, Поступление, Экономика
- [x] **Видео-плейсхолдер** и контент в каждом уроке
- [x] **Мини-тест (квиз)** в каждом уроке с проверкой ответа
- [x] **Прогресс-бар** — % прохождения курса, отмечать уроки выполненными
- [x] **Личный кабинет** — статистика, мои курсы, сохранённые возможности, дедлайны, рекомендации
- [x] **Система рекомендаций** — по интересам ученика при онбординге
- [x] **Админ-панель** — добавить/редактировать/удалить возможности и курсы, список пользователей

### Бонусные функции
- [x] **AI-ассистент** — умный чатбот по возможностям и курсам
- [x] **Календарь дедлайнов** — интерактивный с подсветкой сохранённых дедлайнов
- [x] **Тёмная / светлая тема**
- [x] **3 языка** — Русский / English / Қазақша
- [x] **Сертификаты** — при 100% прохождении курса
- [x] **Roadmap 9–12 классов** — подробный план для поступления в вуз
- [x] **Адаптивный дизайн** — mobile-friendly

## 🛠 Технический стек

| Слой | Технология |
|------|-----------|
| Frontend | Vanilla HTML5 + CSS3 + JavaScript (ES6+) |
| Хранилище | `localStorage` (JSON) |
| Аутентификация | Кастомная (localStorage sessions) |
| Деплой | GitHub Pages / Netlify / Vercel (статика) |
| AI | Rule-based knowledge base (расширяется до Anthropic API) |

### Почему один файл?
- **Нет зависимостей** — не нужен Node.js, npm, сервер
- **Мгновенный деплой** — drag & drop на Netlify
- **Полная функциональность** — всё работает оффлайн
- **Демо-ready** — открываешь и сразу показываешь

---

## 💾 Как работает хранилище (localStorage)

```
mh_users          — массив зарегистрированных пользователей
mh_session        — ID текущей сессии (сохраняется при закрытии)
mh_opps           — массив возможностей (с дефолтными 10 штук)
mh_courses        — массив курсов (с дефолтными 5 штук)
mh_ud_{userId}    — данные пользователя: saved, enrolled, certs
```

## 🗺 Пользовательский путь

```
1. Открыть сайт → Главная страница
2. Зарегистрироваться → выбрать класс, интересы, цели
3. Каталог → фильтры → сохранить возможность ⭐
4. Курсы → записаться → уроки → квиз → прогресс
5. 100% курса → Сертификат 🏅
6. Личный кабинет → дедлайны, прогресс, рекомендации
7. AI-ассистент → вопрос о возможностях
8. Admin (admin@mentoria.kz / admin123) → добавить курс/возможность
```
